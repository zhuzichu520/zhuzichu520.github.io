Detailed Description

FlipLayout 组件是一个自定义的布局组件，允许用户在一系列项目之间进行翻页。它支持水平和垂直方向的翻页，并提供上一页和下一页的控件。

示例用法：

 FlipLayout {
     orientation: Qt.Horizontal
     currentIndex: 0
     decrementItem: Component {
         // 定义上一页的组件
     }
     incrementItem: Component {
         // 定义下一页的组件
     }
     // 添加您的项目
 }

Property Documentation

count : int

组件中的项目总数。


currentIndex : int

当前显示的项目的索引。用户可以通过改变这个索引来切换显示的项目。


decrementItem : Component

定义用于翻到上一页的组件。通常是一个按钮或图标。


incrementItem : Component

定义用于翻到下一页的组件。通常是一个按钮或图标。


orientation : enumeration

设置布局的方向。可以是 Qt.Horizontal（水平）或 Qt.Vertical（垂直）。
