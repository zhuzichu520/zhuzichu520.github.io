Detailed Description

`ToggleButton` 组件是一个具有开关功能的按钮，继承自 `StandardButton`，并扩展了其功能以支持切换状态。按钮的 `highlighted` 状态基于 `checked` 属性，用于指示按钮是否被选中。该组件适用于需要切换状态的交互场景，如开关控件。

示例用法：

 ToggleButton {
  text: qsTr("Toggle Me")
  checked: true
  onClicked: {
      console.log("Button clicked, checked state: " + checked)
  }
 }

Property Documentation

checked : bool

指示按钮是否被选中。按钮的 `highlighted` 状态基于此属性。


highlighted : bool

指示按钮是否被高亮显示。此状态由 `checked` 属性控制。
