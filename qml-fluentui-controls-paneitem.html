Detailed Description

`PaneItem` 组件用于在面板中显示一个项，包含标题、图标、信息徽章等属性。它支持点击事件和右键点击事件，并具有启用状态属性。

示例用法：

 PaneItem {
 title: "Item Title"
 iconSource: "icon.png"
 iconDelegate: IconDelegateComponent
 infoBadge: BadgeComponent
 count: 5
 key: "uniqueKey"
 enabled: true
 __footer: false
 }

Property Documentation

count : int

显示在面板项上的计数值。


enabled : bool

是否启用面板项。默认为 `true`。


iconDelegate : Component

用于显示图标的组件。


iconSource : var

面板项的图标源，可以是图片路径或其他图标数据。


infoBadge : Component

用于显示信息徽章的组件。


key : var

面板项的唯一标识符。


title : string

面板项的标题文本。


Signal Documentation

void rightTap()

面板项被右键点击时发出的信号。

void tap()

面板项被点击时发出的信号。
