Detailed Description

Expander 组件允许用户点击或按下特定按键来展开或折叠内容。它包含用于自定义标题、内容以及控制展开和折叠动画的属性。

示例用法：

 Expander {
     expanded: false
     header: Component {
         // 在此处定义标题组件
     }
     content: Component {
         // 在此处定义内容组件
     }
 }

Property Documentation

content : Component

Expander 的内容组件。该组件在 Expander 展开时可见，折叠时隐藏。


expanded : bool

指示 Expander 是否处于展开状态。如果为 true，内容会展开；否则，内容会折叠。


header : Component

Expander 的标题组件。该组件始终可见，用于控制 Expander 的展开或折叠状态。
