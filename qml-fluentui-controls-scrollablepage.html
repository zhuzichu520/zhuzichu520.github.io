Detailed Description

`ScrollablePage` 组件是一个容器，用于展示具有垂直滚动条的内容。它支持自定义列间距、顶部、左侧、右侧的内边距，以及底部的边距。内容区域被设置为 `Flickable` 以允许用户滚动查看超出视口的内容。

示例用法：

 ScrollablePage {
 columnSpacing: 10
 topPadding: 16
 leftPadding: 16
 rightPadding: 16
 bottomMargin: 16
 content: ColumnLayout {
  // 在此处添加要显示的内容
 }
 }

Property Documentation

bottomMargin : int

页面底部的边距（以像素为单位）。默认为 `24`。


columnSpacing : int

列之间的间距（以像素为单位）。默认为 `0`。


leftPadding : int

页面左侧的内边距（以像素为单位）。默认为 `24`。


rightPadding : int

页面右侧的内边距（以像素为单位）。默认为 `24`。


topPadding : int

页面顶部的内边距（以像素为单位）。默认为 `24`。
