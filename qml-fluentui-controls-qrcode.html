Detailed Description

`QRCode` 组件用于生成和显示二维码，支持自定义二维码的文本、颜色、背景颜色以及二维码的大小和边距。它将二维码绘制在一个矩形背景上，提供了简便的方式来展示二维码。

示例用法：

 QRCode {
 text: "https://www.example.com"
 color: "black"
 backgroundColor: "white"
 size: 100
 margins: 10
 }

Property Documentation

backgroundColor : alias

二维码背景的颜色。


color : alias

二维码的颜色。


margins : int

二维码与背景矩形边缘之间的边距（以像素为单位）。默认为 `0`。


size : int

二维码的大小（以像素为单位）。默认为 `50`。


text : alias

要编码为二维码的文本。
