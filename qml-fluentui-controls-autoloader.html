Properties

  • sourceComponent : Component

Detailed Description

AutoLoader 组件是基于 `Loader` 的封装，用于自动加载和卸载组件内容。在组件销毁时，会自动将 `sourceComponent` 设为 `undefined`，以便释放资源。

示例用法：

 AutoLoader {
     sourceComponent: MyCustomComponent
 }

Property Documentation

sourceComponent : Component

需要加载的组件。AutoLoader 会自动加载该组件，并在销毁时将其设为 `undefined`，以释放资源。
