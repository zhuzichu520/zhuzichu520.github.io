Detailed Description

`RatingControl` 组件用于显示一个评分控件，通常用于用户评分或选择。控件由一系列图标组成，用户可以通过点击或悬停来选择评分。控件支持自定义评分数量、图标大小、间距和当前值。

示例用法：

 RatingControl {
     number: 5
     spacing: 4
     size: 18
     value: 3
 }

Property Documentation

number : int

评分控件中显示的图标数量。默认为 `5`。


size : int

每个图标的大小（以像素为单位）。默认为 `18`。


spacing : int

图标之间的间距（以像素为单位）。默认为 `4`。


value : int

当前评分值。范围从 `0` 到 `number`，默认为 `0`。
