Detailed Description

ColorResource 组件是一个 QtObject，定义了一组颜色属性，这些属性可以在应用程序的不同部分中用于确保一致的颜色主题。该组件将颜色值集中在一个地方进行管理，方便全局访问和修改。

示例用法：

 ColorResource {
  textFillColorPrimary: "black"
  controlFillColorDefault: "white"
  // 其他颜色属性的定义
 }
