Detailed Description

`Typography` 组件定义了一系列字体样式，旨在确保应用中的文本显示一致且符合设计规范。该组件提供了多种字体样式，以满足不同文本内容的需求，如标题、正文、字幕等。

示例用法：

 Label {
  text: qsTr("This is a title")
  font: Typography.title
 }

Property Documentation

body : font

正文文本的基本字体样式，适用于常规的正文内容。字体像素大小为 13，字重为 Normal。


bodyLarge : font

大号正文文本的字体样式，适用于较大段的正文。字体像素大小为 17，字重为 Normal。


bodyStrong : font

强调型正文文本的字体样式，适用于需要突出显示的正文内容。字体像素大小为 13，字重为 DemiBold。


caption : font

说明性文本的字体样式，适用于注释或小型描述文本。字体像素大小为 12，字重为 Normal。


display : font

显示主要文本的字体样式，通常用于重要的标题或大号文本。字体像素大小为 62，字重为 DemiBold。


subtitle : font

副标题的字体样式，适用于次要标题或说明文本。字体像素大小为 19，字重为 DemiBold。


title : font

标题文本的字体样式，适用于普通标题。字体像素大小为 26，字重为 DemiBold。


titleLarge : font

大标题的字体样式，适用于较大的标题文本。字体像素大小为 37，字重为 DemiBold。
