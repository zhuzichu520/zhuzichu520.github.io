Detailed Description

HighlightRectangle 组件用于在用户界面中突出显示某个元素。它支持垂直和水平方向的突出显示，并根据状态变化执行动画。该组件可以用于实现滑块、指示器等效果。

示例用法：

 HighlightRectangle {
     orientation: Qt.Horizontal
     highlightSize: 20
     target: someItem
     highlight: true
 }

Property Documentation

bottomMargin : int

底边距，用于在突出显示时调整矩形的位置。


endOffset : int

突出显示的结束偏移量。


highlight : bool

是否启用突出显示。启用时，矩形会根据状态变化执行动画效果。


highlightSize : int

突出显示区域的大小。


leftMargin : int

左边距，用于在突出显示时调整矩形的位置。


orientation : int

突出显示的方向。可以是 `Qt.Vertical` 或 `Qt.Horizontal`。


startOffset : int

突出显示的起始偏移量。


target : var

突出显示的目标项。只有当目标项具有焦点或满足其他条件时，突出显示矩形才会可见。
