Detailed Description

`Watermark` 组件允许您在应用程序的视图中绘制自定义水印。它提供了多个属性来控制水印的文本、颜色、位置、旋转角度和大小。通过使用此组件，您可以为应用程序的视图添加透明的水印效果，从而增强内容的保护或品牌识别。

示例用法：

 Watermark {
  text: qsTr("Confidential")
  textColor: "red"
  textSize: 30
  rotate: 45
  offset: Qt.point(20, 20)
  gap: Qt.point(10, 10)
 }

Property Documentation

gap : point

水印文本的间隙。此属性定义了文本与其背景之间的距离。


offset : point

水印文本的偏移量。此属性定义了水印相对于其默认位置的偏移量。


rotate : int

水印文本的旋转角度。此属性定义了水印文本的旋转角度，以度为单位。


text : string

水印文本。此属性定义了要在视图中绘制的水印内容。


textColor : color

水印文本的颜色。此属性定义了水印文本的显示颜色。


textSize : int

水印文本的大小。此属性定义了水印文本的字体大小。
