Detailed Description

BannerLayout 组件是一个自定义PathView，设计用于在垂直或水平路径中显示一系列项目。它包含用于控制可见项目数量、自动播放时长和路径方向的属性。

示例用法：

 BannerLayout {
   visibleItemCount: 3
   autoPlayDuration: 2000
   autoPlay: true
   orientation: Qt.Horizontal
   // 在此处添加您的项目
 }

Property Documentation

autoPlay : bool

是否应该自动播放。


autoPlayDuration : int

每个自动播放间隔的时长（以毫秒为单位）。


orientation : int

滑动的方向。使用 Qt.Vertical 或 Qt.Horizontal。


visibleItemCount : int

应在视图中显示的项目数量。
