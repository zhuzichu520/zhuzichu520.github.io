Detailed Description

ContentPage 组件是一个自定义的页面布局，继承自 QtQuick.Controls 的 `Page`。它提供了标题、背景和内边距设置，支持自定义内容的显示。

示例用法：

 ContentPage {
 title: "页面标题"
 // 在此处添加您的内容
 }
