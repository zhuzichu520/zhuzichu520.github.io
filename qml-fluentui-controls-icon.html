Detailed Description

Icon 组件用于显示图标，可以根据 `source` 属性的类型选择显示文本图标或图片图标。组件会根据 `Theme.dark` 状态自动调整颜色，并支持从字体或图片源加载图标。

示例用法：

 Icon {
     source: "icon_name" // 例如 FontAwesome 图标名称或图片路径
     width: 24
     height: 24
 }

Property Documentation

color : color

图标的颜色。根据 `Theme.dark` 状态，颜色会自动调整为白色或黑色。


family : string

字体家族名称，通常用于文本图标。默认为 `FluentIcons.fontLoader.name`。


implicitHeight : int

图标的默认高度。与 `implicitWidth` 相同。


implicitWidth : int

图标的默认宽度。默认为 24 像素。


source : var

图标的源。可以是一个图标名称（文本图标）或一个图片 URL。根据源的类型，组件会选择显示文本图标或图片图标。
