Detailed Description

`RoundClip` 组件用于创建一个圆角矩形裁剪效果，通过使用 `ShaderEffectSource` 和 `OpacityMask` 实现。它允许用户设置圆角半径，并对内部内容应用圆角裁剪效果。

示例用法：

 RoundClip {
 width: 100
 height: 100
 radius: [10, 10, 10, 10]
 content: Rectangle {
  width: 100
  height: 100
  color: "blue"
 }
 }

Property Documentation

content : alias

要应用圆角裁剪效果的内容项。


radius : var

圆角矩形的圆角半径，格式为 `[topLeft, topRight, bottomRight, bottomLeft]`。
