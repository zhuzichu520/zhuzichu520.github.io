Detailed Description

Tools 组件提供了一些常用的系统工具和实用功能，支持跨平台操作。该组件以单例模式存在，可用于处理窗口图标、文件读写、剪贴板操作等。

Property Documentation

systemDark : bool

获取系统是否处于暗色模式。


windowIcon : QUrl

设置或获取窗口图标的 URL。
