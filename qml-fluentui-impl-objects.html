Detailed Description

Objects 组件是一个简单的容器，用于将多个 QML 对象作为子项管理。它提供了一个默认属性，可以将多个 `QtObject` 子项直接添加到该组件中。

示例用法：

 import FluentUI.impl

 Objects {
     id: control
     // 添加多个子项对象
     QtObject { objectName: "child1" }
     QtObject { objectName: "child2" }
 }

Property Documentation

children : list<QtObject>

容器中包含的子对象列表。可以直接在 `Objects` 组件内部添加多个 `QtObject` 子项。
