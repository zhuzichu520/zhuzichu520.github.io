Detailed Description

AccentColor 组件提供了管理和生成一系列颜色亮度的功能，适用于不同的主题环境（如深色模式和浅色模式）。通过 `darkest`、`darker`、`dark`、`normal`、`light`、`lighter` 和 `lightest` 函数，可以根据 `normal` 基础颜色生成一系列的颜色，供其他组件使用。

示例用法：

 AccentColor {
     normal: "#0078D4"
     onCompleted: {
         console.log("Lightest color:", lightest())
         console.log("Default brush for theme:", defaultBrushFor())
     }
 }

Property Documentation

dark : var

深色，可以通过函数 `dark(val)` 设置或获取。如果未设置，则基于 `normal` 颜色生成一个具有 90% 不透明度的颜色。


darker : var

较深的颜色，可以通过函数 `darker(val)` 设置或获取。如果未设置，则基于 `normal` 颜色生成一个具有 80% 不透明度的颜色。


darkest : var

最深的颜色，可以通过函数 `darkest(val)` 设置或获取。如果未设置，则基于 `normal` 颜色生成一个具有 70% 不透明度的颜色。


light : var

浅色，可以通过函数 `light(val)` 设置或获取。如果未设置，则基于 `normal` 颜色生成一个具有 90% 不透明度的颜色。


lighter : var

较浅的颜色，可以通过函数 `lighter(val)` 设置或获取。如果未设置，则基于 `normal` 颜色生成一个具有 80% 不透明度的颜色。


lightest : var

最浅的颜色，可以通过函数 `lightest(val)` 设置或获取。如果未设置，则基于 `normal` 颜色生成一个具有 70% 不透明度的颜色。


normal : var

常规颜色，通常是强调色的基础颜色。可以通过函数 `normal(val)` 设置或获取。


Method Documentation

var defaultBrushFor()

根据当前主题（深色或浅色）返回合适的默认颜色。对于深色主题，返回 `lighter()`；对于浅色主题，返回 `dark()`。


var secondaryBrushFor()

返回一个透明度为 90% 的默认颜色，通常用于次要元素的背景或边框颜色。


var tertiaryBrushFor()

返回一个透明度为 80% 的默认颜色，通常用于三级元素的背景或边框颜色。
