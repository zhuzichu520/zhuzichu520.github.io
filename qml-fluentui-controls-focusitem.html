Detailed Description

FocusItem 组件用于显示当前具有焦点的项的视觉指示。它显示一个边框来突出显示具有焦点的目标组件，边框的颜色和半径可以自定义。

示例用法：

 FocusItem {
     radius: 6
     target: someFocusableItem
 }

Property Documentation

radius : int

边框的圆角半径。


target : Item

用于显示焦点指示的目标组件。只有当目标组件具有焦点时，焦点指示器才会可见。
